HTML table reference

Table tags

Tag Name Description
<table> Table The wrapping element for all HTML tables
<thead> Table Head The set of rows defining the column header in a table
<tbody> Table Body The set of rows containing actual table data
<tr> Table Row Each table row
<td> Table Data Data for each column
<tfoot> Table Footer The set of rows defining the footer in a table

Table Attributes

Attribute Name Description
colspan Column Span
rowspan Row Span Defines how many rows the td element should span
